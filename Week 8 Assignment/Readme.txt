Project Description:
--------------------
This project analyzes global COVID-19 data (cases, deaths, and vaccinations) using Python.
It includes data cleaning, exploratory data analysis (EDA), visualizations, and insights.
The final report is saved as an HTML file for easy viewing.

How to Use:
-----------
1. Open the Jupyter Notebook inside "notebooks/" to view or rerun the analysis.
2. To quickly view results, open "reports/covid_analysis.html" in a web browser.
3. Dataset is available inside "data/" (owid-covid-data.csv).

Dependencies:
-------------
- Python 3.x
- pandas
- matplotlib
- seaborn
- (Optional) plotly

Notes:
------
- `.ipynb_checkpoints/` folders are automatically created by Jupyter and can be ignored.
- The `figures/` and `scripts/` folders are empty (reserved for optional use).
